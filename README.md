# PREDICTING-VEHICLE-FUEL-EFFICIENCY-USING-LINEAR-REGRESSION-MODELS
Developed regression models to predict MPG using the Auto MPG dataset. Compared Simple, Multiple, and Polynomial architectures while implementing Lasso and Ridge shrinkage to prevent overfitting. Validated via LOOCV and K-Fold CV, optimizing the RMSE vs. Adjusted R^2 trade-off to ensure robust model generalization.
